# 👁️ Code Oracle (Голос Оракула)
**Персональная облачная среда для вайбкодинга и визуальной разработки.**

---

## ЧАСТЬ 1. ТЕХНИЧЕСКИЙ ПАСПОРТ (Архитектура)

**Назначение системы:**
Code Oracle — это веб-интерфейс, который объединяет искусственный интеллект, редактор кода и файловую систему GitHub. Он позволяет разрабатывать программные продукты (например, проекты «Живое Таро» или «ZORKI AI») через естественный диалог, не написав ни строчки кода вручную.

**Как устроены узлы связи:**
1. **Мозг (ИИ):** Нейросеть Google Gemini 2.0 Flash. Общение с ней идет через универсальный шлюз OpenRouter. Модель настроена выдавать строгие, полные блоки кода.
2. **Память (GitHub API):** Приложение напрямую читает и записывает данные в репозитории на GitHub. Для авторизации используется секретный токен `GITHUB_PAT`.
3. **Хранилище браузера (Local Storage):** Интерфейс автоматически запоминает имя владельца (`marataitester-blip`) и название репозитория.
4. **Визуальный Алтарь (Редактор):** * Текстовые файлы открываются в профессиональном редакторе Monaco Editor.
   * Медиафайлы (картинки `.png/.jpg/.webp`, аудио `.mp3`, видео `.webm/.mp4`) открываются напрямую через стриминг ссылок из GitHub, не перегружая систему.

## ЧАСТЬ 2. РУКОВОДСТВО ПО ЭКСПЛУАТАЦИИ (Мануал)

Этот алгоритм — твой ежедневный цикл работы. Следуй ему шаг за шагом.

### Шаг 1. Подключение к Хроникам (Инициализация)
1. Открой приложение в браузере.
2. Убедись, что в левом верхнем углу заполнены поля:
   * **Владелец:** твой логин на GitHub (например, `marataitester-blip`).
   * **Репозиторий:** название проекта (например, `Tarot_new` или `code-oracle`).
3. Нажми синюю кнопку **«ПОДКЛЮЧИТЬ ХРОНИКИ»**.
4. *Результат:* Внизу слева загрузится дерево всех файлов твоего проекта.

### Шаг 2. Выбор объекта (Что меняем?)
* **Если файл существует:** Кликни на него в дереве файлов. Его содержимое (или изображение/видео) мгновенно появится в правом окне.
* **Если нужен новый файл:** Нажми зеленую кнопку **«+ НОВЫЙ ФАЙЛ»** в дереве слева. Напиши путь (например, `src/components/Card.tsx`) и нажми галочку. Справа появится пустой холст.

### Шаг 3. Разговор с Оракулом (Разработка)
1. В строке чата слева внизу напиши свою инженерную задачу.
   * *Пример:* «Оракул, добавь в этот файл кнопку "Вытянуть Карту Дня"».
2. Дождись ответа ИИ. Он выдаст готовый блок кода.
3. Внимательно нажми кнопку **«ПРИМЕНИТЬ КОД К РЕДАКТОРУ»** под ответом ИИ.
4. *Результат:* Код мгновенно перепрыгнет в правое черное окно.

### Шаг 4. Материализация (Сохранение в GitHub)
1. Посмотри на правое окно. Если тебя всё устраивает, нажми верхнюю зеленую кнопку **«МАТЕРИАЛИЗОВАТЬ (PUSH)»**.
2. Дождись всплывающего окна "Материализация успешна!".
3. *Результат:* Твой новый код физически сохранен в GitHub. Система готова к следующему шагу.
